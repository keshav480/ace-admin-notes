Simple draggable admin notes overlay. Create/edit notes in the admin, and they persist per site.

== Description ==
Adds a floating canvas on admin screens where authorized users can place sticky notes. Notes are saved as a site option and can be shown/hidden via the admin bar.

== Installation ==
1. Upload to `/wp-content/plugins/`
2. Activate from Plugins.
3. Manage notes under the “Ace Notes” menu.
