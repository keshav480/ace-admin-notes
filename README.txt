A lightweight draggable admin notes plugin that lets you create sticky notes directly in the WordPress admin area.

== Description ==

**Ace Admin Notes** adds a floating, draggable notes canvas to the WordPress admin area, allowing authorized users to create, edit, move, and manage sticky notes directly on admin screens.

Notes are stored securely as site options and persist across admin pages, making it easy to leave reminders, to-do items, or internal notes for yourself or your team.

Features include:

* Floating draggable notes overlay in the admin
* Create, edit, and delete notes easily
* Notes persist across admin screens
* Show or hide notes using the admin bar
* Notes saved per site (not per browser)
* jQuery UI powered drag & resize
* Lightweight and performance friendly
* WordPress.org review friendly

This plugin is ideal for site administrators, developers, and teams who want quick reminders or internal notes inside the WordPress dashboard.

== Installation ==

1. Upload the `ace-admin-notes` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Go to **Ace Notes** from the admin menu.
4. Create and manage your notes.
5. Use the admin bar toggle to show or hide notes.

== Frequently Asked Questions ==

= Who can see the admin notes? =

Only authorized users with access to the WordPress admin can see and manage the notes.

= Are notes saved permanently? =

Yes. Notes are saved as a site option and persist until deleted.

= Can I hide the notes temporarily? =

Yes. You can show or hide the notes overlay using the admin bar toggle.

= Do notes appear on the frontend? =

No. Notes are visible only in the WordPress admin area.
